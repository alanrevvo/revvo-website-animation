# Animation Flow

This is a visual explanation of Revvo's tire platform

Click on the dots!

## Installation
- `npm install`

## Local Testing
- `npm run build && ( sleep 3 && open 'http://localhost:8080' & ) && npm start`
- Also, check out `localhost:8080/message.html`

## Dependencies
- jquery
- underscore
