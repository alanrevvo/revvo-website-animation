# Animation Flow

This is a demo that used to run on Medallia's engineering website.

Click on the dots!

## Installation
- `npm install`

## Local Testing
- `npm run build && ( sleep 3 && open 'http://localhost:8080' & ) && npm start`
- Also, check out `localhost:8080/message.html`

## Dependencies
- jquery
- underscore
